# Агуулахын систем

Системд бараа нь эхлээд Product хэлбэрээр бүртгэгдэнэ. Бараа агуулахад орж ирэх үед Income үүсэж, түүнд харгалзах Receipt болон ReceiptItem-ууд бүртгэгдэнэ. Үүний үр дүнд тухайн барааны үлдэгдэл InventoryReport дээр нэмэгдэнэ.

Бараа агуулахаас гарах үед Expense үүсэж, тухайн барааны үлдэгдэл InventoryReport дээр хасагдана.

Агуулахын бүх мэдээллийг Warehouse class удирдах бөгөөд үүнд бараа, орлого, зарлага, нярав зэрэг мэдээллүүд хадгалагдана. Storekeeper (нярав) нь барааг орлогоор хүлээн авах, зарлага гаргах, үлдэгдэл шалгах, тайлан гаргах болон тооллого хийх үйлдлүүдийг гүйцэтгэнэ.

Систем нь дараах тайлангуудыг гаргана:
  ->InventoryReport – тухайн үеийн барааны үлдэгдэл
  ->StorekeeperReport – тодорхой хугацааны тайлан (эхний үлдэгдэл, орлого, зарлага, эцсийн үлдэгдэл)

Мөн барааны тоо хэмжээг шалгах үед StockCount ашиглан тооллого хийж, системийн үлдэгдэлтэй харьцуулан зөрүүг тодорхойлно. Зөрүү гарсан тохиолдолд барааны үлдэгдлийг шинэчилнэ.
